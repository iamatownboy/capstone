# RAG 및 개인정보보호(PII) 평가용 가상 비즈니스 데이터셋 (Synthetic Dataset)

본 디렉토리의 데이터셋은 **RAG(Retrieval-Augmented Generation) 시스템의 개인정보 유출 탐지 및 필터링 성능 평가**를 위해 알고리즘으로 자동 생성된 **가상의 합성 데이터(Synthetic Data)**입니다.

---

## ⚠️ 중요 고지 및 면책 조항 (Synthetic Data Disclaimer)
- **가상 데이터 선언**: 본 데이터셋 내의 모든 임직원명, 이메일 주소, 전화번호, 주민등록번호, 외국인등록번호, 여권번호, 운전면허번호, 카드번호, 은행 계좌번호, 주소, IP 주소, 소속 부서 및 회사 기밀 정보 등은 **Faker 라이브러리 및 Luhn 알고리즘 등을 기반으로 난수 생성된 가상의 정보**입니다.
- **실제와의 무관성**: 본 데이터셋에 등장하는 이름, 연락처, 식별 정보 등은 실제 인물, 단체, 기업, 혹은 실제 서비스 환경의 데이터와 어떠한 연관성도 없으며 우연히 일치하더라도 이는 100% 무작위 생성 결과에 따른 것입니다.
- **활용 목적**: 본 자료는 RAG 파이프라인 상에서 개인정보 보호 필터링 장치가 정상적으로 오차 없이 동작하는지 테스트하는 **평가/학습 목적 외의 용도로는 사용을 금합니다.**

  - 임직원 연봉 계약서, M&A 검토서, 고객 개인정보 명부, 인사 평가, 보안 사고 침해 보고서 등 민감한 개인식별정보(PII) 및 기업 대외비가 포함된 문서입니다.

### 📊 지원하는 개인식별정보(PII) 카테고리 (33종)
본 합성 데이터는 실제 보안 가이드라인에 규정된 33종의 PII 데이터 템플릿(성명, RRN, 카드번호, 계좌번호, 이메일, IP 주소 등)이 비즈니스 맥락 속에 골고루 분포되어 있어 정밀한 오검출(False Positive)/미검출(False Negative) 진단이 가능합니다.

### 📝 다변화된 15가지 문서 양식
텍스트 데이터가 일률적인 형태를 띠지 않고, 자연스러운 검색이 가능하도록 15가지 고유 레이아웃에 결합되어 생성됩니다:
1. **REPORT (보고서)**: 요약문 및 추진 마일스톤 타임라인 테이블 포함
2. **EMAIL (이메일)**: 3턴 왕복 스레드(Threaded Reply Top-Posting) 구현
3. **SLACK (슬랙)**: 10~12턴의 이모지 피드백 포함 다자간 대화록
4. **WIKI (사내위키)**: 최신 개정 버전 이력 Markdown 테이블 및 관련 문서 링크 포함
5. **MINUTES (회의록)**: 회의 진행 아젠다 및 참석자별 발언록 상세 포함
6. **COMMIT_LOG (Git 커밋)**: Git Diff 형태의 소스코드 변경 블록 포함
7. **DEV_LOG (개발일지)**: 단위 테스트 결과 및 Jest 터미널 실행 트레이스 포함
8. **ASSET_LIST (자산대장)**: 배정 임직원 정보가 기재된 8-10행 규모의 테이블
9. **STATUS_CHECK (상태보고)**: 서버 리소스 메트릭 수치 및 시간별 이벤트 로그 포함
10. **QA_BOARD (QA게시판)**: VPN, OTP 등 접속 관련 댓글 토론 스레드 포함
11. **VOC_TICKET (VOC티켓)**: 상담원-고객 1:1 라이브 채팅 대화 및 상담 녹취록 포함
12. **CONTRACT (계약서)**: 손해배상 조항 및 대표이사(인), 주소 정보가 포함된 계약 정보 체결 영역 포함
13. **AUDIT_LOG (감사로그)**: 시스템 위반 예외 스택 트레이스 및 감사 이벤트 원시 JSON 데이터 포함
14. **INVOICE (지출청구서)**: 단가, 수량, 세금(부가세 10% 자동 계산)이 포함된 품목 단가 테이블 포함
15. **RESUME_HR (인사카드)**: 이전 직장 이력 및 다면 평가 점수 패널 포함
